# Global Energy and Economic Analysis

## Overview
This project analyzes the relationship between global energy consumption, economic growth, and scientific research contributions across top-ranked countries.

The analysis combines multiple datasets related to energy supply, GDP, and journal contributions to identify patterns, correlations, and country-level performance indicators using Python-based data analytics techniques.

## Business Objective
Understanding the relationship between energy usage, economic performance, and research output can support:
- Economic and sustainability analysis
- Policy and investment decisions
- Resource allocation strategies
- Global development benchmarking

## Dataset Sources
The project combines publicly available datasets related to:
- Global energy supply
- GDP and economic indicators
- Scientific journal contributions

## Methodology

### 1. Data Cleaning and Preprocessing
- Handled missing values and inconsistencies
- Standardized country names and formats
- Removed irrelevant records and duplicates

### 2. Data Integration
- Merged multiple datasets into a unified analytical dataset
- Joined datasets using country-level identifiers

### 3. Feature Engineering
Created additional analytical metrics and calculated fields to support deeper analysis.

### 4. Exploratory Data Analysis (EDA)
Performed:
- Trend analysis
- Distribution analysis
- Correlation analysis
- Country-level comparisons

### 5. Visualization
Created visualizations to communicate:
- GDP vs energy consumption
- Scientific output comparisons
- Correlation patterns
- Country performance trends

## Tools and Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Identified strong relationships between GDP and energy consumption across leading economies.
- Observed patterns linking scientific output with economic development.
- Highlighted differences in energy efficiency and research contributions between countries.
- Generated data-driven insights through visualization and correlation analysis.
